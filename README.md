## ✨ Features

- 🖼️ **Thumbnail Editor** — Design eye-catching thumbnails with a drag-and-drop layout editor
- 📦 **Product Photo Builder** — Generate polished product photos ready for e-commerce
- 🎨 **AI Scene Generation** — Describe a scene and let Gemini AI create a matching background
- ✂️ **Background Removal** — Cleanly remove image backgrounds with one click
- 🔄 **Reimaginer** — Transform and reimagine existing images using AI
- 🕐 **History** — Browse and restore your previously generated creations
- ⚙️ **Settings** — Customize theme and app preferences
- 🌙 **Dark / Light Mode** — Full theme support

## 🚀 Getting Started

### Prerequisites

- **Node.js** v18 or higher
- A **Google Gemini API key** — get one free at [Google AI Studio](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/AbhinavTheDev/Super-Banana.git
   cd Super-Banana
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**

   Create a `.env.local` file in the project root and add your Gemini API key:

   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Start the development server**

   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:5173`.
